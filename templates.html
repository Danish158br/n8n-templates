Templates Kahan se Milenge?

n8n ke bohot saare ready-made templates online available hain. Aap inko download karke apne n8n instance mein import kar sakte hain.

GitHub Repositories:

GitHub per bohot se developers ne apne banaye hue n8n workflows ko JSON files ki shakal mein share kiya hua hai. Kuch mashoor repositories yeh hain:

  • **wassupjay/n8n-free-templates**: Ismein 200 se zyada plug-and-play workflows hain.
  • **simealdana/ai-automation-jsons**: AI se related automation workflows ke JSONs.
  • **enescingoz/awesome-n8n-templates**: n8n automation templates ka curated collection.

Templates ko Kaise Istemal Karein (Import Karna)

Templates ko istemal karna bohot aasan hai:

  1. **JSON File Download Karein:** GitHub repository se ya n8n Community Hub se apni pasand ke workflow ki `.json` file download karein.
  2. **n8n mein Import Karein:**
    • Apne n8n instance mein login karein.
    • Left sidebar mein "Workflows" per click karein.
    • Upar right side per "New" ya "Add Workflow" button ke paas "Import from File" ka option hoga, us per click karein.
    • Download ki hui JSON file ko select karein aur import kar lein.
  3. **Configure Karein:** Import karne ke baad, workflow aapke canvas per load ho jayega. Ab aapko usmein apne credentials aur zaroori settings (jaise email addresses, API keys) configure karni hongi.
  4. **Test aur Activate Karein:** Settings theek karne ke baad, workflow ko test karein aur phir activate kar lein.

Kuch Ahem Templates ki Misaalein

Yahan kuch bunyadi aur aam istemal hone wale templates ki misalein di gayi hain. Inka JSON structure dekh kar aapko idea ho jayega ke yeh kaise kaam karte hain.

Example 2: Email Notification on New RSS Feed Item

Yeh workflow kisi RSS feed mein naye item aane par aapko email bhejta hai.

{
  "nodes": [
    {
      "parameters": {
        "url": "https://www.n8n.io/blog/rss.xml"
      },
      "name": "RSS Feed Read",
      "type": "n8n-nodes-base.rssFeedRead",
      "typeVersion": 1,
      "id": "rss-feed-read-node",
      "position": [250, 300]
    },
    {
      "parameters": {
        "fromEmail": "your-email@example.com",
        "toEmail": "your-email@example.com",
        "subject": "Naya RSS Feed Item: {{ $json.title }}",
        "text": "Naya item publish hua hai: {{ $json.link }}"
      },
      "name": "Send Email",
      "type": "n8n-nodes-base.emailSend",
      "typeVersion": 1,
      "id": "send-email-node",
      "position": [500, 300]
    }
  ],
  "connections": {
    "RSS Feed Read": {
      "main": [
        [
          {
            "node": "Send Email",
            "type": "main",
            "index": 0
          }
        ]
      ]
    }
  },
  "active": false,
  "id": "rss-to-email-workflow",
  "name": "RSS to Email Notification",
  "staticData": null,
  "settings": {},
  "createdAt": "2025-07-30T10:00:00.000Z",
  "updatedAt": "2025-07-30T10:00:00.000Z",
  "versionId": null
}
